saya Bintang

Saya saat ini sedang menjalani praktek di dunia kerja di Dinas Komunikasi dan Informatika (Kominfo). Sebagai bagian dari praktek ini, tugas utama saya adalah membuat proyek portofolio tentang diri saya. Dalam proyek ini, saya mengumpulkan dan mengorganisir informasi mengenai pengalaman akademik dan profesional saya, keterampilan teknis dan non-teknis, serta berbagai proyek yang pernah saya kerjakan.
Proyek portofolio ini mencakup pembuatan halaman web yang menampilkan profil pribadi saya, daftar riwayat hidup, dan dokumentasi proyek-proyek yang pernah saya selesaikan. Saya menggunakan berbagai teknologi web seperti HTML, CSS, dan JavaScript untuk membuat tampilan yang menarik dan fungsional. Selain itu, saya juga belajar tentang pentingnya presentasi visual dan bagaimana mengkomunikasikan informasi dengan efektif melalui media digital.
Melalui tugas ini, saya tidak hanya mengasah keterampilan teknis saya dalam pengembangan web, tetapi juga mendapatkan wawasan yang lebih dalam tentang bagaimana cara mempresentasikan diri secara profesional di dunia kerja. Pengalaman ini sangat berharga dan membantu mempersiapkan saya untuk karir masa depan di bidang teknologi informasi dan komunikasi
